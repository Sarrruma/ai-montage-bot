# Telegram AI Montage Bot

Бот для автоматизации рекламы с AI-оптимизацией.

## Функции
- Генерация уникальных ссылок с UTM-метками
- Автозамена креативов через AI
- Интеграция с Make.com + Google Sheets

## Развертывание
1. Установи зависимости: `pip install -r requirements.txt`
2. Настрой BOT_TOKEN в переменных окружения
3. Запусти: `python bot.py`
